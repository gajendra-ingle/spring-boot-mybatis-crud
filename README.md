# Spring Boot + MyBatis CRUD Application

I created this project to practice and understand how to build a simple CRUD (Create, Read, Update, Delete) application using Spring Boot and MyBatis.  

The goal of this project is to:

- Learn how Spring Boot works with REST APIs  
- Understand MyBatis integration with a database  
- Practice connecting a Java application to MySQL  
- Implement basic CRUD operations  
- Follow a clean layered architecture (Controller → Service → Mapper)

This project helps strengthen backend development fundamentals.

---

## 📂 What This Repository Contains

This repository includes:

- A Spring Boot application setup
- REST API endpoints for CRUD operations
- MyBatis mapper interface and XML configuration
- MySQL database connection configuration
- User entity model
- Service layer for business logic
- Controller layer for handling HTTP requests
- Maven configuration (`pom.xml`)
- Application configuration file (`application.yml`)

The project demonstrates a simple and clean structure for building backend applications using Spring Boot and MyBatis.
